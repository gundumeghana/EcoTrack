# 🌱 EcoTrack – Smart Waste Management System

## 🚀 Overview

EcoTrack is a smart waste management web application designed to track, monitor, and improve plastic waste collection and recycling efficiency. It provides real-time insights through an interactive dashboard and helps optimize waste handling processes.

---

## 🛠️ Tech Stack

* ⚛️ React (TypeScript)
* 🔥 Firebase (Database & Backend)
* 🎨 CSS / Tailwind (UI Styling)
* ⚡ Vite (Build Tool)

---

## ✨ Features

* 📊 Real-time waste tracking dashboard
* ♻️ Recycling efficiency monitoring
* 📍 Waste hotspot visualization
* 🧾 Complaint management system
* 🚛 Collection tracking system
* 🔄 Dynamic data updates using Firebase

---

## ⚙️ Run Locally

### 1. Clone the repository

```bash
git clone https://github.com/gundumeghana/EcoTrack.git
cd EcoTrack
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start development server

```bash
npm run dev
```
